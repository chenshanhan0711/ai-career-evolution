# 中国就业结构与招聘数据来源

## 年度就业结构数据

长期演化与预测使用世界银行开放数据 API 中的中国年度指标，原始统计口径为 ILO
模型估计：

- 第一产业就业人员占比：`SL.AGR.EMPL.ZS`
- 第二产业就业人员占比：`SL.IND.EMPL.ZS`
- 第三产业就业人员占比：`SL.SRV.EMPL.ZS`
- 国家/地区代码：`CHN`
- 覆盖年份：1991—2025 年，共 35 个年度
- 获取脚本：`scripts/fetch_employment_trends.py`
- 派生文件：`china_employment_structure.csv`
- 文件 SHA-256：`05a5e073da37692ad768a693dc472385127002bbec805702914bfcace19a604f`

API 示例：<https://api.worldbank.org/v2/country/CHN/indicator/SL.SRV.EMPL.ZS?format=json&per_page=100>

该年度序列用于描述中国三次产业就业结构，不直接测量生成式AI的因果影响。平台使用
最近 12 年线性趋势进行 2026—2030 年外推，并以五年滚动回测残差构造区间。

## 招聘岗位数据

本项目的分析数据来自 Kaggle 数据集 **BOSS_Zhipin_Sample_Data**：

- 页面：https://www.kaggle.com/datasets/jeanshendev/boss-zhipin-sample-data
- 发布者：jeanshendev
- 平台版本更新时间：2025-04-18
- 许可证：MIT（以数据集页面标注为准）
- 原始文件：`BOSS_Zhipin_Sample_Data.xlsx`
- 原始记录：109,986 条，16 个字段，373 个发布城市
- 原始压缩包 SHA-256：`89eef2fbd2c932465067c794604f7d6b0a3b65a33de8bf43285190cbdcff99e7`
- 原始 XLSX SHA-256：`4b247747ac476782c26b0cfb84861076fc019251d1cbd9e709708eeff401b10a`

仓库中的 `china_jobs_clean.csv.gz` 是由 `scripts/prepare_real_dataset.py` 从原始
XLSX 清洗得到的可复现派生文件。处理包括字段重命名、职位链接去重、薪资单位换算、
职业类别规则映射和 AI 关键词标记。未生成或补造招聘记录。

## 数据边界

该数据是招聘平台公开样本，不是实时 API，也不是概率抽样。多数城市约包含 300 条记录，
因此城市记录数不能直接解释为城市总体招聘规模。项目使用城市样本的薪资中位数、AI
岗位占比、学历结构和职业构成进行描述性比较，并明确标注为样本统计。

职业类别的产业结构情景指数以 2025 年为 100，将当前招聘样本中的行业构成映射到三次
产业预测。该指数不是岗位数量预测，也不表示失业率、职业消失概率或 AI 替代概率。

## 岗位文本代理指标

平台从职位名称、关键词和行业字段识别数字化、重复任务、创造性任务与人际互动术语，
按职业类别计算命中率，并在13个类别内进行相对缩放。AI任务暴露代理指数由数字化强度
45%、直接AI术语30%和“数字化强度×重复性”25%组成。该指标用于课程中的相对比较，
不是官方AI暴露统计、自动化概率或裁员风险。

职业转型路径把13类当前职业映射到AI增强目标岗位，转型准备度由数字化强度40%、创造性
30%和AI暴露代理30%组成；目标岗位与能力缺口采用明确的情景规则。路径不表示已经发生的
人员迁移，页面同时保留当前高频岗位关键词作为数据依据。
